|  Code  |  含义 |  备注  |
| --- | --- | --- |
| 20 | 车型| 参考车维度字典接口获取可选值 |
| 30 | 厂商 |参考车维度字典接口获取可选值 |
| 35 | 品牌 | 参考车维度字典接口获取可选值 |
| 40 | 子级别 |  参考本页**车维度-子级别 level**枚举 |

| 50 | 国别 | 参考本页面国别枚举 |

## 时间类型 dateType
|  Code  |  含义 |  备注  |
| --- | --- | --- |
| 10 | 周 | 按周粒度进行查询，例如开始时间2021-11-01，结束时间2021-11-07，开始时间必须是周一，结束时间必须是周日 |
| 20 | 月 | 按月粒度进行查询，例如2021-11 |
| 40 | 年 | 按年粒度进行查询，例如2021 |
